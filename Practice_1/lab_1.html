Задание 1.

Вычислить кол-во комбинаций из n различных элементов по k. Количество комбинаций определяется рекуррентным соотношением:
Ckn = n! / k! * (n - k)!, если n > 0
Ckn = 1, если k = n >= 0 или k = 0, n < 0
Ckn = 0, в остальных случаях

Код программы с комментариями:

	(define (fact n) ; процедура нахождения факториала числа 

		  (if (= n 0) 

		      1 

		      (* n (fact (- n 1))))) 


	(define (comb n k) ; процедура нахождения кол-ва комбинаций 

	  (cond ((or (and (= n k) (> n 0)) (and (= k 0) (< n 0))) 

	         1) 

	         ((and (> n 0) (> n k)) 

	          (/ (fact n) (* (fact k) (fact (- n k))))) 

	         (else 0) 

	         ) 

	  ) 

	

	(comb 10 2) ; вызов процедуры

Задание 2.

Используя рекурсию, вывести все числа от А до В включительно, в порядке убывания, если А больше В или в порядке возростания в другом случае. Контрольный тест: введенные числа 7 и 3, полученный результат: 7, 6, 5, 4, 3.

Код программы с комментариями:

(define (func a b) ; процедура вывода чисел в порядке возростания/убывания 

  (display a) 

  (display " ") 

  (cond ((> a b) 

        (func (- a 1) b)) 

        ((< a b) 

         (func (+ a 1) b)))) 



(func 7 3) ; вызов процедуры
